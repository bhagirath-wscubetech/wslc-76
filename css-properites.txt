3) To change the font size

4) font-weight: bold | normal
5) text-decoration: underline | line-throught | overline | none

6) border: size color type (all direction)
   border-top: size color type
   border-left: size color type
   border-right: size color type
   border-bottom: size color type
   
	eg: size: 10px , 5px
	eg: color: red , yellow
	eg: type: solid , dotted, dashed, double

7) margin is space outside the border
	margin: 5px (all direction)
	margin-top: 5px (top)
	margin-left: 5px (left)
	margin-right: 5px (right)
	margin-bottom: 5px (bottom)
	margin: 10px 5px; (y-axis x-axis) 
		top-bottom: 10px
		left-right: 5px
	margin:auto (x-axis)

	padding-right: 5px (right)

	padding: 10px 5px; (y-axis x-axis)
